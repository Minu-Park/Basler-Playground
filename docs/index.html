CORE FEATURE SHOWCASE

Auto-Generated JIT Interactive Parameters

Add simple // @interactive annotations in code to construct live GUI parameter controls instantly without restarting image acquisition.

How It Works

  1. 1
    Annotate Script Code

    Add // @interactive-range(min, max) next to any variable in Script Editor.

  2. 2
    Auto-Generated GUI Controls

    Basler Playground parses annotations and constructs real-time parameter sliders.

  3. 3
    Zero-Latency Pipeline Update

    Drag sliders to tune thresholding, edges, or gain dynamically without dropping frames.

Live Interactive Parameter Hot-Swapping Latency: 0.2ms 
// Dynamic algorithm script
auto input = get().toMatGray();
auto output = cv::Mat();


int range = 127; // @interactive-range(1, 255)

cv::threshold(input, output, range, 255,
              cv::THRESH_BINARY);

show(output);
